[筆記]Laravel開發環境建置

Feb 26, 2018

Laravel提供快速方便的環境建置方式-Homestead

Homestead預設環境

  • Ubuntu 14.04
  • Git
  • PHP 5.6 / 7.0
  • Nginx
  • MySQL

  • Composer

    ……

安裝Homestead

首先下載/安裝好VirtualBox或是VMWare，之後在安裝Vagrant。
接下來安裝Vagrant Box，在終端機執行下列指令，通常會需要一段時間可以先去睡zzz

下載Homestead的Vagrant設定檔，用Git clone下來，並執行init

設定Homestead檔

在剛剛clone下來的Homestead資料夾裡有個Hometestead.yaml，它是用來設定環境的配置

  • ip: 設定local端的IP位置

  • memory: 配給虛擬機記憶體的大小

  • cpus: 配給虛擬機CPU的個數

  • provider: Vagrant的提供者

  • authorize: ssh key位置

  • folders: 設定共享目錄

  • sites: 設定網域與共享目錄

  • port: 轉port設定

啟動Vagrant囉~

在Homestead目錄下輸入指令啟動Vagrant
